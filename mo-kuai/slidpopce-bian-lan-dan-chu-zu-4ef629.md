# slidpop

侧边栏弹出组件

### API

```
   /**
     * 打开
     * @param url 弹出页面的js地址
     * @param param 参数
     * @param delt 宽度或者高度
     * @param offset  边距
     * @param side 上下左右(left,top,right,bottom)
     */
   public void show(String url, HashMap param,float delt,HashMap offset,String side)


    /**
    *关闭
    */
    public  void dismiss()
```
